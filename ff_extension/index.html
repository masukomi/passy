PasswordHash Firefox Extension

About this project

ThumbnailCopy adds a right-click entry to fill in passwords based on a master password and a domain. Install it here!

About this project: This is a reasonably simple Firefox extension written in response to this blog entry.

Interesting source files:

  • content/firefoxOverlay.xul - the overlay that has the window and menu entries.
  • content/overlay.js - the JavaScript file that does the password calculations.
